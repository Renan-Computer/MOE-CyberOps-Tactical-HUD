# MOE-CyberOps-Tactical-HUD
Interface tática de alto desempenho para guerra cibernética e monitoramento de espectro. Projetada com ergonomia militar, oferece modo de visão noturna, TSD (Tactical Situation Display) para mapeamento de ameaças e interface otimizada para uso com luvas. Focada em operações de campo, prioriza visibilidade crítica e segurança de dados sensíveis.
